# アーカイブ

このディレクトリには、完了したタスクのドキュメントがアーカイブされます。

## アーカイブ方針

- タスクが完了したら、`ongoing/` から対応するディレクトリをこのディレクトリに移動
- アーカイブ時には、完了日や成果物のサマリーを記録
- 必要に応じて参照できるよう、適切な命名規則で管理

## 命名規則

完了したタスクは、以下の形式でアーカイブすることを推奨します：

```
archived/
└── {タスク名}-{完了日}/
    └── （タスク関連のドキュメント）
```

例: `rules-compliance-migration-2024-12-31/`
