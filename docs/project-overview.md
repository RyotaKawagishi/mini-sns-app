# プロジェクト概要

## プロジェクト名
mini-sns-app

## 目的
SNSアプリケーションのサンプル実装

## 主要機能

### ユーザー管理
- ユーザー登録・認証
- アカウント有効化（メール認証）
- パスワードリセット
- プロフィール編集
- ユーザー一覧・詳細表示

### マイクロポスト
- マイクロポストの投稿・削除
- 画像アップロード
- リプライ機能（in_reply_to）
- フィード表示

### フォロー機能
- ユーザー間のフォロー・アンフォロー
- フォロー中・フォロワー一覧表示
- フォロー中ユーザーの投稿をフィードに表示

## 技術スタック

### フレームワーク・言語
- Ruby 3.2.6
- Rails 7.1.5

### データベース
- SQLite3 (開発・テスト)
- PostgreSQL (本番)

### フロントエンド
- Bootstrap 3.4.1
- Stimulus (Hotwire)
- Turbo (Hotwire)

### その他
- Active Storage (画像アップロード)
- Action Mailer (メール送信)
- BCrypt (パスワードハッシュ化)

## 現在の状態

### アプリケーションタイプ
- **Webアプリケーション**: HTMLビューを返す従来型のRailsアプリ
- **API**: 現時点ではAPIエンドポイントは提供していない

### テストフレームワーク
- Minitest
- Fixtures

### 認証・認可
- セッション認証
- コントローラー内での認可ロジック実装

## 今後の方向性

### ルール準拠への移行
- コーディング規約（`.cursor/rules/`）への準拠
- テストフレームワークの移行検討（RSpec）
- 認可ロジックの分離（Pundit）
- API設計ツールの導入検討（Blueprinter, Committee Rails）
