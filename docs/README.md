# Documentation

このディレクトリには、プロジェクトのドキュメントが含まれています。

## 構成

### プロジェクト全体で共有するドキュメント
- `project-overview.md` - プロジェクトの全体像、目的、主要機能
- `components/` - コンポーネント別の詳細ドキュメント
  - `controllers.md` - コントローラーの設計と実装方針
  - `models.md` - モデルの設計とドメインロジック
  - `routes.md` - ルーティング設計
  - `testing.md` - テスト戦略と方針

### 進行中のタスク
- `ongoing/` - 現在進行中のタスクや作業に関するドキュメント
  - `rules-compliance-migration/` - ルール準拠への移行タスク

### 完了したタスク
- `archived/` - 完了したタスクのアーカイブ

## 使用方法

これらのドキュメントは、AIアシスタントがプロジェクトの文脈を理解し、適切なコード生成やリファクタリングを行うために使用されます。

## 更新方針

- プロジェクト全体で共有するドキュメントは、プロジェクトの重要な変更時に更新
- 進行中のタスクは `ongoing/` に配置し、完了後は `archived/` に移動
- 各タスクは独立したディレクトリとして管理
